# CONDOMINIO WEB APP FRONTEND

Este proyecto esta construido

- Nextjs
- Axios
- Material UI

## Enviroment

Debe asegurarte de tener node 14 o superior instalado y luego instalar los modulos de node

```bash
yarn
```

Ahora para levantar el proyecto

```bash
yarn dev
```
